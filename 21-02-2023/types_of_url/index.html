URL(Uniform Resource Locator) is the path to the desired file, document or web page

Generally the URL is of two Types:
1) Absolute URL
2) Relative URL

Absolute URL:
Absolute URL consists of the full destination address of the destination document including the proxy, server and path of the document.

Relative URL:
Relative URL contains only the partial source address, it acts as an endpoint over the current source url and it can be only accessed if the destination document is also on the same server as source document.
