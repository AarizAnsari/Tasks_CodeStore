Types Of Formatting In HTML

Formatting Element Use Example
<b> Used to Bold the text This text is bold
<i> Used to display text in Italic This text is Italic
<strong> Used to define importamt text This text is Important
<em> Used to provide emphasis on the text Put emphasis on this text
<small> Used to reduce text size This is a small text
<mark> Used to mark the text This text is marked
<del> Used to show deleted text The year is 2022 2023
<ins> Used to show inserted text The year is 2022 2023
<sub> Used to display subscript elements Chemical formula of water is H2O
<sup> Used to display superscript elements 52 = 25
