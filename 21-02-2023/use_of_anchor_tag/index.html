A static web page to demonstrate the use of <a> tag

<a> reffers to the anchor tag.
There are several attributes of the anchor tag:

  1. href is used to redirect the user to another page.

  2. download is used to download the desired file.

  3. hreflang is used to specify the language of the linked page.

  4. media is used to define for what device the url page is optimized

  5. ping is used to define all urls the browser will notify if the hyperlink is clicked

  6. referrerpolicy is used to define the referr information of the source

  7. referrerpolicy is used to define the referr information of the source

  8. rel attribute is used to define the relation between the source and destination documents

  9. target is used to define the mode of display of the target page

  10. type is used to define the type of the target document
