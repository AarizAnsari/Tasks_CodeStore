What is HTML?

HTML stands for HYPER TEXT MARKUP LANGUAGE.
HTML acts as a basic skeleton of the web page and provides a basic structure to it.
HTML used different tags to add different types of components such as buttons, text, images, links etc.

To use image, img tag is used and a source is provided

To use Button, button tag is used

To add a Link, a(anchor) tag is used
