USE OF HTML TAGS

HTML uses tags to build the structure of the web page.
Everything in the html is enclosed inside the html tag.

Different types of html tags are :

  1. HTML:

    Opening and closing tag for every html page.

  2. Head:

    Meta data holder tag.

  3. Body:

    Holds all the data that is to be displayed over the page.

  4. Title:

    Defines the page title.

  5. P(Paragraph):

    Used to write Text Paragraphs.

  6. H1(Heading):

    Used to write heading where the size can be selected by using tags between h1 to h6.

  7. Ol(Ordered List):

    List where the list items are in a serial numeric order.

  8. Ul(Unordered List):

    List where the list items are in a bullet point order.

  9. Dl(Definition List):

    List where the list items are provides with an added definition.

  10. Div(Division):

    To divide web page to add a border or limit for components.

  11. Img(Image):

    To add an image to the web page.

  12. Button:

    To add a button to the web page.

  13. A(Anchor):

    To add a hyperlink to the web page.

  14. Picture:

    It is used to add different resolution images to improve responsiveness.

  15. Hr(Horizontal):

    It is used to add a horizontal partition to the page.

  16. Form:

    It is used to add a form to the page.

  17. Label:

    It is used to label to a form text field.

  18. Input:

    It is used to read the data from the form text field.
